# JS Play - JavaScript Learning Folder

This repository is a personal JavaScript practice space for learning, experimenting, and building small examples.

## Goals

- Practice core JavaScript concepts
- Try out small coding exercises
- Build confidence with problem solving

## How to Use This Folder

1. Add topic-based files/folders (for example: `basics/`, `arrays/`, `functions/`, `dom/`).
2. Write short examples and experiments.
3. Run files with Node.js:

```bash
node your-file.js
```

## Suggested Learning Topics

- Variables and data types
- Operators and conditionals
- Loops
- Functions
- Arrays and objects
- ES6+ features (arrow functions, destructuring, spread/rest, modules)
- DOM basics (if using browser JavaScript)
- Async JavaScript (`setTimeout`, promises, `async/await`)

## Notes

Keep this repo simple and consistent. Focus on learning by writing and testing code often.
